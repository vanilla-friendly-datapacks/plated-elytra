# Plated Elytra

> A Minecraft 1.16.x data pack that adds support for plating elytras with armor to enhance their armor values.

## Installation

Download the latest release from [here][latest] and put the datapack zip file into the `datapacks` folder of your Minecraft world. Download the resourcepack zip file (if available) and enable the pack in your client.

I usually provide resourcepacks to allow translations of my datapacks :)

## Crafting

Drop an elytra together with a chestplate of your choice and a slime ball on top of an anvil.

> **Note:** Chainmail chestplates are not supported and this is intentional.
